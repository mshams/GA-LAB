# GA-LAB
 Genetic Lab problem solver
 Supported selection methods:
  - Roulette
  - Tournament Rnd
  - Tournament Unique
  - Ranking Linear
  - Ranking Exp
  - Ranking Bias1
  - Ranking Bias2
  - Ranking NoLinear
  - Boltzman
  - Scaling Linear
  - Scaling Exp
  - Truncation
		  
 Supported crossover methods:
  - Random
  - 1Point
  - 2Point
  - Uniform
  - GST
  - GSX
  - Greedy1pt
  - Greedy2pt
  - PMX
  - Circular
  - NoRepeat
  ...
